# ✈️ Ducksplorer — Travel Planner

A full-stack MERN travel planning web app that allows users to plan trips, book hotels and activities, manage budgets, and receive real-time notifications.

---

## ✨ Features

- Browse and book hotels, flights, and activities
- Real-time booking management and availability
- Budget tracker with spending breakdowns
- User authentication and profile management
- Notifications system for booking updates
- Complaints and feedback module
